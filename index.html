    🤖 Inteligência artificial

    Chave gratuita em aistudio.google.com → Get API key. Sem cartão de crédito. Obs.: no tier gratuito, o Google pode usar os prompts para melhorar os modelos.

    Funciona com qualquer serviço no padrão OpenAI. Para o Grok, crie a chave em console.x.ai (a API da xAI é paga, por créditos). Se o serviço bloquear chamadas do navegador (CORS), veja a alternativa do Cloudflare Worker no README.
